Enunciado: Você foi contratado para desenvolver a simulação de um portão automático de duas partes para a entrada de uma empresa. Esse portão será representado por dois servo motores, que deverão se movimentar para simular a abertura e o fechamento. Além disso, o sistema contará com três LEDs para indicar o seu status: vermelho para portão fechado, amarelo para portão em movimento e verde para portão aberto.

Análise: Um portão automático de duas partes (cada uma sendo um servo motor), 3 LEDs de cores diferentes (vermelho, amarelo e verde).

Encontrar uma solução: A luz verde deve acender apenas quando os portões estiverem totalmente abertos, a luz amarela deve acender quando os portões estiverem em movimento (abrindo e fechando), a luz veremlha deve acender somente quando os portões estiverem fechados.

Implementar uma solução: Enquanto o portão estiver fechado, a luz vermelha estará acesa, em um intervalo de 15 segundos; enquanto estiver se movendo para abrir, deverá acender a luz amarela; quando estiver aberto, deve acender a luz verde, em um intervalo de 15 segundos; quando estiver se movendo para fechar, deve acender a luz amarela.

Avaliação da solução: Em situações de vida real, essa solução funciona.
